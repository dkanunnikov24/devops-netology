# devops-netology

В файле .gitignore в каталоге terraform для git будут проигнорированы следующие файлы и папки:

- Директории /.terraform/
- Файлы у которых в названии встречается .tfstate
- Файл crash.log и файлы по шаблону crash.*.log
- Файлы с расширением .tfvars
- Файлы override.tf и override.tf.json а также файлы по шаблонам *_override.tf и *_override.tf.json
- Конкретные файлы и папки:
.terraformrc
